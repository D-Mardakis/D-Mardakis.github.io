Start of tables

HTML tables allow web developers to arrange data into rows and columns. The table tag is used to define the table.
The tr tag is used to create table rows and the td tag, which stands for table data, is used to create table cells.
The th tag, which stands for table header, creates table cells like td, but by default the text in th elements are bold and centered.
